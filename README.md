"# Campus Jobs Platform" 
"" 
"Платформа для поиска временной работы и стажировок в университетском кампусе." 
"" 
"Описание проекта" 
"" 
"Сервис централизованной платформы для поиска временной работы и стажировок в кампусе университета." 
"" 
"Технологии" 
"" 
"- **Бэкенд**: Django 5.2, Python 3.12" 
"- **База данных**: MySQL" 
"- **Фронтенд**: HTML, CSS, JavaScript, Bootstrap 5" 

"Описание проекта" 
"Модели базы данных:"
**User** - стандартная модель Django"
**StudentProfile** - профиль студента"
**EmployerProfile** - профиль работодателя"
**Category** - категория вакансии"
**Skill** - навыки"
**Vacancy** - вакансия/стажировка"
**Application** - заявка на вакансию"
**Interview** - собеседование"
**Review** - отзыв"
**Notification** - уведомление"
**VacancySkill** - связь вакансии и навыков"
**StudentSkill** - связь студента и навыков"
